# index.css

Global reset and base typography for the entire application.

## Purpose
Standardizes fonts, box-sizing, and body styles across the SPA.

## Key Features
- **Root Variables**: Defines global colors and easing functions.
- **Inter Font**: Sets `Inter` as the primary font-family.
- **Base Reset**: Ensures `box-sizing: border-box` and resets margin/padding on common elements.
- **Global Background**: Provides a subtle background gradient for the entire application.
