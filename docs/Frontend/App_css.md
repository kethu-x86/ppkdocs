# App.css

The global stylesheet for the root application container.

## Purpose
Sets the default layout constraints for the React application, including the main `#root` container's max-width, centering, and basic padding.

## Key Styles
- **`#root`**: Limits the width to 1280px and centers the application.
- **`.logo`**: Defines the styling and hover effects for the React and other logos.
- **`@keyframes logo-spin`**: Handles the rotation animation for the React logo.
- **`.card`**: Provides basic padding for card-like containers.
- **`.read-the-docs`**: Sets a muted color for documentation-related links.
