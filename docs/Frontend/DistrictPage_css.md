# DistrictPage.css

Styles for the activity discovery and itinerary building interface.

## Purpose
Manages the layout for `DistrictPage.jsx`, featuring district-specific hero sections and activity card grids.

## Key Styles
- **Hero Background**: Full-width imagery with overlay effects for district-specific branding.
- **Activity Cards**: Hover-responsive cards with icons, labels, and expandable info sections.
- **District Grid**: Optimized grid layout for displaying a wide range of activities.
- **Interactive Elements**: Styling for info-toggle buttons and selection checkboxes.
