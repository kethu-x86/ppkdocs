# PaymentPage.css

Styles for the secure payment simulation interface.

## Purpose
Supports the UI in `PaymentPage.jsx`, featuring credit card forms and success states.

## Key Styles
- **Payment Card**: Centered card with glass effects and secure styling.
- **Card Form**: Optimized layouts for card number, expiry, and CVV inputs.
- **Success State**: Large icon and green-themed confirmation messaging.
- **SSL Badge**: Styling for security indicators and trust signals.
