# AdminDashboard.css

Styles for the admin-only control panel.

## Purpose
Supports the UI in `AdminDashboard.jsx` for managing hotel inventory and availability.

## Key Styles
- **Control Panel**: Layout for the main administrative interface.
- **Inventory List**: Grouped and filtered list of all hotels across Kerala.
- **Status Indicators**: Color-coded badges and pills indicating "Active" or "Disabled" status for each stay.
- **Filter Bar**: Styling for the district-based filtering dropdown.
- **Action Buttons**: Standardized primary and secondary buttons for administrative tasks.
