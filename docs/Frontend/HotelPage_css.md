# HotelPage.css

Styles for the hotel search results and listing interface.

## Purpose
Manages the aesthetic of `HotelPage.jsx`, featuring grouped results based on activities.

## Key Styles
- **Hotel Card**: Detailed card layout with rating badges and budget status indicators.
- **Budget Badge**: Color-coded badges for "Best Match" (green) and "Premium" (red) status.
- **Activity Groups**: Clearly separated sections for hotels near specific activities.
- **Commute Box**: Styling for the estimated travel time and distance indicators.
- **Modals**: Styling for the budget alert dialog shown when no stays match the planner.
