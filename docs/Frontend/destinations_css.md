# destinations.css

Styling for the main destination discovery and trip planning interface.

## Purpose
Provides visual styles for the `Login.jsx` (trip planner view) and other destination-centric UI elements.

## Key Styles
- **Planner Layout**: Grid and row-based layouts for the multi-field travel planning form.
- **Search Bar**: Modern styling for the header search input.
- **Planner Inputs**: Standardized aesthetics for select boxes and date pickers used in the planner.
- **Animations**: CSS animations for staggered entry and fade-in effects.
