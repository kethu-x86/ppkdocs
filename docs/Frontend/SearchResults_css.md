# SearchResults.css

Styles for the global search results page.

## Purpose
Supports the UI in `SearchResults.jsx`, featuring split sections for destinations and hotels.

## Key Styles
- **Search Header**: Centered title and prominent back button for results.
- **Results Grid**: Multi-column grid optimized for both destination and hotel cards.
- **Dropdown Menu**: Styling for the room-selection dropdown that appears when a hotel is clicked.
- **Shimmer Effect**: CSS animations for loading placeholders (shimmer cards) during the search fetch.
- **Modal Overlay**: Specific styling for the trip-planning modal shown when a destination is selected.
- **Book Now**: Prominent green action buttons for direct hotel booking.
