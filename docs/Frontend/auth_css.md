# auth.css

Shared styling for authentication pages and login forms.

## Purpose
Provides consistent styling for `Login.jsx`, `AdminLogin.jsx`, and overall layout of authentication cards.

## Key Features
- **Teal Color Palette**: Defines primary, dark, and light teal variants used throughout the application.
- **Glass-morphism Effects**: Uses semi-transparent backgrounds and blurs for UI cards.
- **Layout**: Centered authentication cards with a split-screen design (form on left, hero content on right) for larger viewports.
- **Form Styling**: Standardizes input fields, labels, and primary action buttons.
