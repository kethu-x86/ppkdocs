# RoomPage.css

Styles for the room selection and nearby exploration view.

## Purpose
Manages the aesthetic of `RoomPage.jsx`, featuring detailed room cards and interactive exploration modals.

## Key Styles
- **Room Card**: Card layouts with high-quality room imagery, ratings, and amenity meta-info.
- **Quantity Selector**: UI for choosing the number of rooms based on guest count.
- **Explore Modal**: Comprehensive modal layout for viewing nearby amenities (Petrol, EV, Food, etc.).
- **Budget Alerts**: Styling for budget warnings when room prices exceed the user's limit.
