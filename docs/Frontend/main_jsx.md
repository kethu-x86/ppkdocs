# main.jsx

The React application entry point.

## Purpose
Bootstraps the React application into the DOM's `#root` element.

## Key Functions
- **`createRoot`**: Initializes the React 18/19 root.
- **`StrictMode`**: Wraps the main `App` component for development checks.
- **Import Hub**: Imports global styles (`index.css`) and the main `App` component.
