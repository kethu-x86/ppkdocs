# BookingPage.css

Styles for the final booking summary and customization page.

## Purpose
Manages the layout and aesthetics of the `BookingPage.jsx` component, including the facility selection grid and the budget tracking UI.

## Key Styles
- **Budget Tracking**: Custom styling for the progress bar and labels that reflect real-time spending.
- **Facility Grid**: Interactive card layouts for selecting various room amenities.
- **Price Breakdown**: A clearly defined footer area summarizing all costs before the final grand total.
- **Modals**: Styling for budget-exceeding warning dialogs.
