# LandingPage.jsx

The initial application entry and welcome screen.

## Purpose
Provides a visually appealing, high-impact welcome screen to the "Personalized Travel Planning System."

## Key Features
- **Hero Image**: Uses a travel-themed background image for visual appeal.
- **Clear CTA**: A prominent "Arrow Forward" button to enter the main application.
- **Glass-morphism Overlays**: Uses semi-transparent backgrounds to ensure text readability over the hero image.
- **MUI Components**: Built using `@mui/material` for a consistent, modern design.
- **One-Click Entrance**: Quickly transitions the user to the `Login` page via the `onEnter` prop.
