# FacilitiesPage.css

Styles for the additional facility selection modal/view.

## Purpose
Supports the UI in `FacilitiesPage.jsx` and `BookingPage.jsx` for selecting amenities.

## Key Styles
- **Fac-Grid**: Responsive grid layout for facility icons and prices.
- **Checkbox Box**: Custom-styled checkbox indicators for selection status.
- **Header/Footer**: Consistent styling for facility-related action areas.
- **Card Styling**: Rounded corners and subtle shadows for facility cards.
