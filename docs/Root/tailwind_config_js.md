# tailwind.config.js

Tailwind CSS configuration file.

## Purpose
Customizes the Tailwind CSS framework for the project, including theme extensions and content scanning.

## Key Features
- **Content Paths**: Directs Tailwind to scan `index.html` and everything in the `src` directory for class names.
- **Theme Extensions**: Allows for custom color palettes (like the teals used in this project) and font extensions.
- **Plugins**: Includes `autoprefixer` for cross-browser CSS compatibility.
