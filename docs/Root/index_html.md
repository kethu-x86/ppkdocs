# index.html

The single-page application's main HTML entry.

## Purpose
Provides the root structure for the React application to mount.

## Key Elements
- **`<div id="root"></div>`**: The target element for the React DOM.
- **`<script type="module" src="/src/main.jsx"></script>`**: Loads the React application entry.
- **`<meta name="viewport">`**: Ensures the application is responsive on mobile devices.
- **`<title>`**: Sets the browser tab title.
