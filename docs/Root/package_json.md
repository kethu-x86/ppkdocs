# package.json

The central project manifest and dependency tracker.

## Purpose
Defines project metadata, development scripts, and all third-party dependencies required for the React application.

## Key Scripts
- **`npm run dev`**: Starts the Vite development server.
- **`npm run build`**: Compiles the application for production.
- **`npm run lint`**: Runs ESLint to check for code quality issues.
- **`npm run preview`**: Previews the production build locally.

## Major Dependencies
- `react` & `react-dom` (v19): The core UI library.
- `@mui/material`: Used for pre-built, modern UI components.
- `react-router-dom` (v6/v7): Handles client-side routing.
- `framer-motion`: Provides smooth animations and staggered entry effects.
- `tailwindcss`: Utility-first CSS framework (v4).
