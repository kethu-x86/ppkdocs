# postcss.config.js

PostCSS transformation configuration.

## Purpose
Sets up PostCSS to process CSS with plugins like Tailwind CSS and Autoprefixer.

## Plugins
- **`@tailwindcss/postcss`**: Integrates Tailwind directly into the CSS pipeline.
- **`autoprefixer`**: Automatically adds vendor prefixes for cross-browser support.
