# vite.config.js

The configuration file for the Vite build tool.

## Purpose
Sets up the development and build environment for the React application.

## Key Configurations
- **`@vitejs/plugin-react`**: Enables fast refresh and optimized React compilation.
- **Root Resolution**: Sets the project root to the correct path.
- **HMR**: Configures Hot Module Replacement for a fast development experience.
