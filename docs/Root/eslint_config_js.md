# eslint.config.js

ESLint configuration for code quality and consistency.

## Purpose
Ensures the JavaScript and React code follows best practices and avoids common errors.

## Rules Used
- **`@eslint/js`**: Provides the base recommended JavaScript rules.
- **`eslint-plugin-react-hooks`**: Ensures proper usage of `useState`, `useEffect`, etc.
- **`eslint-plugin-react-refresh`**: Helps with the Hot Module Replacement developer experience.
